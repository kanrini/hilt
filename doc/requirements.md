* ## Hilt 핵심 요약

  ### 1. 정체성

  - SWORD 엔진 기반 초경량 CLI 성경 리더.
  - C23 + libsword (C++ 브릿지).
  - **타겟:** 리눅스 (특히 아치에서 써먹을거).

  ### 2. 기능 (Functional)

  - **스마트 조회:** 버전 안 치면 기본값(`~/.hilt/config`)으로 출력.
    - 한국어 약어(`요 3:16`) 완벽 지원.
  - **범위/장 조회:** `1-14` 절 범위나 `7:` 장 전체 조회 가능.
  - **리스트 모드 (`-l`):** 본문 말고 절 번호만 10개씩 끊어서 나열.
  - **설정 (`-s`):** 선호하는 성경 버전 설정.

  ### 3. 출력 스타일

  - **Clean:** XML/OSIS 태그 같은 쓰레기 제거.
  - **ANSI Color:** 주소는 Cyan, 에러는 Red. 가독성이 생명.

  ### 4. 데이터

  - **외부 모듈:** 데이터는 로컬(`~/.sword`)에 있는 거 쓰기.
